# pyabpoa (TeloXplorer Fork)
This is a custom fork of the [abPOA](https://github.com/yangao07/abPOA) (v1.5.6) project, originally developed by Yan Gao (yangao@ds.dfci.harvard.edu).

**Modifications:**
This version has been specifically modified to support a 128-character ASCII alphabet. It bypasses the standard DNA/Amino Acid translation tables to allow for the Multiple Sequence Alignment (MSA) of highly tokenized Telomere Variant Repeats (TVRs).

This fork is maintained strictly as a backend alignment dependency for the [TeloXplorer](https://github.com/hhuili/TeloXplorer) pipeline.
